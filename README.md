# Face Expression Detector

A real time face expression detector using webcam built with Python.

## Emotions Detected
Happy, Sad, Angry, Fear, Surprise, Disgust, Neutral

## How To Run

Clone your project
  git clone https://github.com/YOURNAME/face-expression-detector

Go into folder
  cd face-expression-detector

Install libraries
  pip install -r requirements.txt

Run
  python face.py

## Requirements
- Python 3.11
- Webcam

## Built With
- Python
- OpenCV
