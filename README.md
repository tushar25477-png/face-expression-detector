# Face Expression Detector

A real time face expression detector using webcam built with Python.

## Emotions Detected
Happy, Sad, Angry, Fear, Surprise, Disgust, Neutral
## How To Run

Step 1 - Install Python 3.11 from python.org

Step 2 - Download the project:
git clone https://github.com/tushar25477-png/face-expression-detector

Step 3 - Go into the folder:
cd face-expression-detector

Step 4 - Install all libraries:
pip install -r requirements.txt

Step 5 - Run the program:
python face.py

Step 6 - Press Q to quit

## What Gets Installed
- opencv-python
- deepface
- tensorflow
- tf-keras
- numpy
- Pillow

## Note
First run will be slow because
DeepFace downloads AI models 
automatically. After that it 
will be fast.
